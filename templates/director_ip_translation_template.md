# Visual Atmosphere Translation Template

Use this template when a user names a familiar animation, studio, art-history, or recognizable visual atmosphere.

```markdown
# 氛围转译：{传播版名称}

## 1. 商用安全版转译
{商用安全版中文} / {commercial_safe_english}

## 2. 它真正的视觉 DNA
- 构图：{composition}
- 色彩：{color}
- 光影：{lighting}
- 材质 / 笔触：{texture}
- 镜头语言：{camera}
- 情绪：{emotion}
- 典型元素：{motifs}
- 禁止项：{avoid}

## 3. 推荐风格组合
{safe_label} 70% + {support_style} 20% + {accent_style} 10%

## 4. 中文提示词
{commercial_safe_chinese_prompt}

## 5. English Prompt
{commercial_safe_english_prompt}

## 6. 负面提示词
不要复制具体角色、Logo、专有场景、知名道具；不要廉价化、不要通用模板感、不要过度 AI 光斑、不要风格混乱。

## 7. 质检
- 是否能看出“氛围 DNA”，而不是只像普通通用风格？
- 是否避开了具体 IP 角色和专有设计？
- 色彩、构图、材质、镜头语言是否统一？
```
