## 3. 15s 视频（模型路由）

**节拍表**：0–3s 雨夜窗外霓虹虚焦 → 3–6s 焦点移到茶杯热气 → 6–9s 侧脸剪影入画 →
9–12s 手指轻触杯壁特写 → 12–15s 拉远定格品牌落版

**路由**：情绪氛围镜头 → 即梦/Seedance（无对白，不需要音画同出）

**主提示词（52 词）**：
```
雨夜茶馆窗边，霓虹光斑虚焦闪烁，焦点缓慢移向一杯冒热气的茶，一个侧脸剪影
缓缓入画坐下，手指轻触杯壁，镜头缓慢后拉定格全景。胶片颗粒，红绿混合光，
慢速运镜，固定水平线，无台词，无背景音乐。
```

**换模型改法**：换 Veo 3 → 追加声音层「雨声轻拍玻璃，茶水轻晃声，无台词，低音量爵士 BGM」。

**负向运动控制**：不要跳切、不要瞬移、不要主体变形、不要粒子乱飞、不要文字乱码。
