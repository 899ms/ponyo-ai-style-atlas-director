# Example — 宫崎骏氛围转译为六宫格故事板

## 传播版参考
宫崎骏风格 / 吉卜力氛围

## 商用安全版
日系手绘治愈动画风

## 主题
《风经过小镇》——一个孩子在乡间小镇追着一阵风，发现风把遗失的信件带回了老邮局。

## 六宫格分镜

1. 远景：蓝天白云下的山坡小镇，红屋顶、绿草地、风吹过晾衣绳。
2. 中景：孩子骑着自行车经过田野，风把一封旧信从草丛里卷起。
3. 特写：信封在阳光里翻飞，纸张泛黄，手绘纹理清晰。
4. 跟拍：孩子穿过小桥和溪流，风吹动树叶和云影。
5. 中远景：老邮局门口，老人抬头，信封轻轻落在门阶上。
6. 收尾远景：小镇黄昏，孩子和老人站在邮局前看云，风铃轻响。
