# Example — 视觉氛围转译海报

## 目标
做一张 3:4 竖版宣传海报，展示 Skill 的 V2 价值：不是给通用风格词，而是把熟悉的视觉氛围转译成商用安全视觉语言。

## 核心转译

1. 宫崎骏风格 → 日系手绘治愈动画风  
2. 皮克斯风格 → 温润家庭向 3D 动画电影感  
3. 迪士尼风格 → 华丽童话城堡动画感  
4. 梵高风格 → 后印象派厚涂旋涡笔触  
5. 莫兰迪风格 → 低饱和灰调静物美学  

## 海报提示词

3:4 vertical premium editorial poster, parchment and ink-wash Chinese art-book layout, title “PONYO AI Style Atlas Director”, subtitle “视觉氛围转译 Skill”, central five collectible style cards, each card shows a commercial-safe visual translation: Japanese hand-painted healing animation countryside, warm family-friendly 3D animation interior, ornate fairytale castle animation, post-impressionist impasto swirling sky, muted gray-toned still-life palette. Elegant gold borders, refined serif typography, subtle mountains and bamboo shadows, clean luxury layout, no copyrighted characters, no logos, no exact IP scenes, premium creative-direction poster.

## 质检重点

- 不能只像普通“AI 风格图谱”；必须体现“氛围转译”。
- 五张卡片要像五种清晰的视觉语言，而不是随机贴图。
- 文字层级要像产品发布海报：主标题、价值声明、五个能力点。
